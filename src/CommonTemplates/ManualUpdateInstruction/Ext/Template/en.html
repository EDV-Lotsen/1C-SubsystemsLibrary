Installing configuration updates

To install a configuration update, follow these steps:

  1. We recommend that you select a time slot when a minimum number of users are working with the application.
  2. Terminate all user sessions.
  3. Terminate your own session.
  4. Start 1C:Enterprise in Designer mode.
  5. In Designer, execute the Update database configuration command located in the Configuration menu (or press F7).
  6. Close Designer.
  7. Start 1C:Enterprise and resume your work.
