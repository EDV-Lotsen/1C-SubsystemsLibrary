Tasks are used as a vehicle for user communication and business process automation.

To create a task, you can utilize a business process, such as Job. To create a task based on an object, use the Create based on submenu. The selected object will become the task subject.

Task notification settings

To ensure notifying performers of any new tasks, you need to enable the New performer task notifications scheduled job which sends new task notifications by email on a regular basis. The email messages are sent to addresses specified in contact information available in the Users catalog. To set up a scheduled job, go to the Scheduled and background jobs window.
