Additional attributes and data are used in property sets. Any additional attributes or data can be displayed in reports or lists.

When choosing between additional properties please refer to these guidelines:

  • Additional attributes are considered to be an integral part of objects with properties. They are stored in the tabular section of the object and can be accessed by any user allowed to access the object with properties itself. The additional attributes are edited in the object window, similar to other attributes.
  • Additional data is used to store object data not considered to be an integral part of the object. Generally, additional data of the object can be viewed or edited even by users not allowed to access the object with properties itself. The additional data are edited in a separate window accessible from the window of object with properties.

When selecting Additional value or Additional value (hierarchy) data type, you can specify a list of allowed values or values with hierarchy for an additional attribute or data. For this, execute Create command from additional attribute or data window.
