Custom fields and data are used in property sets. Any custom fields or data can be displayed in reports or lists.

When choosing between additional properties please refer to these guidelines:

  • Custom fields are considered to be an integral part of objects with properties. They are stored in the tabular section of the object and can be accessed by any user allowed to access the object with properties itself. The custom fields are edited in the object window, similar to other attributes.
  • Custom data is used to store object data not considered to be an integral part of the object. Generally, custom data of the object can be viewed or edited even by users not allowed to access the object with properties itself. The custom data are edited in a separate window accessible from the window of object with properties.

When selecting Additional value or Additional value (hierarchy) data type, you can specify a list of allowed values or values with hierarchy for an custom field or data. For this, execute Create command from custom field or data window.
