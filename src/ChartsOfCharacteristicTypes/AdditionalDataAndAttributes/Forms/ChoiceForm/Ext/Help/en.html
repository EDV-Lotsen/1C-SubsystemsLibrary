This list is used to select additional attributes (data).

It is accessible from Sets of additional attributes (data) list via Add - By sample or Add - Shared command.

Adding values by sample

You can reuse any additional attributes and data from other sets, or add multiple attributes of the same type that share a common value list (example: Door color attribute based on Color sample attribute). In this case, value list of the sample attribute will be used. Preparing value lists for multiple attributes of the same type with a single action saves time and efforts.

  • Select an additional attribute (data) and click Select.
  • Or double-click the item.
  • Type and all values of the additional attribute (data) set as sample will be copied to the new item.

Adding common additional attributes (data)

An attribute is considered "common" when it is included in multiple sets. This option is used when it is necessary to share both an attribute and its value list. In lists and reports of catalog/document items, the common attributes are grouped together and displayed as a single aggregated attribute in a separate column or field. Example: Color common attribute can be used to display all clothes and shoes items of yellow color.

  • Select an additional attribute (data). If the required attribute is not available in the list, click Create. Fill in the required fields.
  • Click Select.
  • Or double-click the item.
