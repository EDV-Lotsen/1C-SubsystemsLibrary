This list is used to select contact information kinds.

To access this list, click Select in the new contact information field input window.

Contact information kinds are predefined groups that include lists or documents containing contact information. Items in each group define content of contact information for the related applied objects. Group content cannot be modified. The group list is a hierarchical tree.

Selecting contact information kinds

  • Select a group and click Select.
