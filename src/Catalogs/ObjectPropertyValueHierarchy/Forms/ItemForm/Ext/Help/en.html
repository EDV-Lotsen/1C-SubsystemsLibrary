This form is used to enter values of custom fields (data).

You can work with custom fields (data) of multiple types. Data type is selected in settings of custom fields (data). When you select Additional value (hierarchy) data type, you are also prompted to enter data values. For each custom field or data of this type, a separate value list is used.

Entering values

  • Enter Description of an custom field (data) value.
  • In the Included in group field, you can specify that the current item is subordinate to another item from the list. To create a new superior item, click Create (or press F8). To specify an existing item, click Show all (or press F4) and then select an item from the list.
  • If Use weight coefficients for values check box is selected for an custom field (data), enter numerical value in Weight coefficient field.
    • This field can be useful for analytical and reporting purposes. You are free to decide how you prefer to use weight coefficients in your analytics and reports. For example, weight coefficients for "Reliability" attribute can be managed in this way: set weight coefficients for attribute values 1-3 to zero; set weight coefficients for attribute values 4-5 to 1. This technique is useful for fast selection of reliable customers with positive weight coefficients.
