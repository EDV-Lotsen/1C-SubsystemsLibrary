This list is used to store values of additional attributes (data).

You can work with additional attributes (data) of multiple types. Data type is selected in settings of additional attributes (data). When selecting Additional value (hierarchy) data type, you are also prompted to enter data values for this list. For each additional attribute or data of this type, a separate value list is used.

Additional value (hierarchy) data type is commonly used; prompting users to select a value from the list expedites data input and reduces the input error rate. This data type can be used to describe various properties of applied objects.

Value hierarchy can be applied to this list. Items of this list have identical properties but they can have subordinate items. The hierarchy structure can become very complex.

You can select any item in the hierarchy as a value, while in the grouped list of additional values, selecting a group name as a value is not permitted.

Selecting values

  • Select an item and click Select.
  • Or double-click the item.

Entering values

Click Create to enter values. Fill in the required fields.
