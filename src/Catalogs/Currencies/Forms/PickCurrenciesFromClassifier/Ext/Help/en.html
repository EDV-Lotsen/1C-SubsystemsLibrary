This form is used to store general currency information and to add new currencies. This is a recommended method to add currencies, as it ensures correct currency data input.

The Currency classifier includes:

  • Numeric code - a unique numeric ID of a currency. Complaint with international standards. It is used to identify currencies when downloading exchange rate data from the Internet sources.
  • Alphabetic code - brief name of a currency. Compliant with international standards.
  • Currency name - full name of a currency.
  • Countries and territories - a list of countries and territories worldwide where this currency is used as primary currency.
  • Import from Internet - select this check box to allow downloading currency data from the Internet sources.

Adding currencies from classifier to currency list

  • Select one or several currencies, and click Select.
  • Or double-click a currency to add it to the currency list.
