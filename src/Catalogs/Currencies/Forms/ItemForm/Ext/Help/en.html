Currency properties description.

Currencies are commonly used in routine corporate activities: all documents related to finances (invoices, money orders, agreements, bills, and more) contain the Currency field, explicitly or by default. The Currency field stores amounts in a variety of currencies (US dollars, foreign currencies, or unspecified "currency units"), depending on the financial geography of your company.

To convert amounts in foreign currencies to US dollars, exchange rates are used.

General currency information

When adding a currency from the Currency classifier, the following fields are filled in automatically:
  • Currency name - full name of the currency.
  • Numeric code - a unique numeric ID compliant with international standards. It is used to identify currencies when downloading exchange rate data from the Internet sources.
  • Alphabetic code - a brief currency name compliant with international standards.
  • Download from Internet - select this check box to allow downloading currency data from Internet sources. When adding a currency from the Currency classifier, this check box will be selected automatically.

Currency-in-words parameters

To ensure that currency amounts expressed in words are correctly displayed in print forms, it is necessary to specify the "currency-in-words" parameters for both integer and fractional parts of the number.

  • Integer part - specify declension parameters for the integer part: singular and plural forms of the currency name.
  • Fractional part - specify declension parameters for the fractional part: singular and plural forms of the fractional currency name.
  • Fraction digits count - if the fractional part is displayed on print forms as number, you can set the number of digits to be displayed on document print forms. For example, if Fraction digits count is set to 1, the amount will be displayed as 150.2 Euro. If set to 2, the amount will be displayed as 150.20 Euro.
  • Amount in words test - the Amount field value will be displayed in this field using the currency-in-words parameters. Use this option to check if your currency-in-words parameters work correctly.

The default currency-in-words prarameters are filled for the following currencies: Dollar and Euro. You can fill it for other currencies as needed.

Interdependencies between currency rates

This functionality can be useful, for example, when you need to calculate an internal currency rate for your company.

  • To calculate interdependent rates, clear the Download from Internet check box.
  • Select the Currency rate depends on other currency rates check box.
  • Select Primary currency from list, or type it in. Primary currency is the currency used to calculate exchange rates for dependent currencies.
  • Specify Margin, %. This is the ratio between exchange rate of the primary currency and exchange rate of the dependent currency.

Viewing exchange rates

Click Currency rates on the currency navigation panel. For your viewing convenience, the date interval is adjustable.

Entering currency rates manually

The primary currency rates are available for download from online sources of financial data. If data for a currency is unavailable for download, the Download from Internet check box will be disabled. In this case, exchange rate data must be filled in manually.

  • Click Currency rates on the currency navigation panel.
  • Click Create and fill in the exchange rate for the selected date (today by default).
