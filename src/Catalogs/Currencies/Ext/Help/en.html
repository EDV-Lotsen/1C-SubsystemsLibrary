The Currencies list stores currencies that you use for transactions.

The Currencies list is only available if the Multicurrency check box is selected in Accounting settings.

To add a currency

  1. In the Lists section, click Currencies and then click Create.
  2. Enter or select the following:
    • Currency code. The unique currency identifier, for example EUR.
    • Symbol. The currency symbol, for example €.
    • Default A/R account. The default A/R account for transactions in this currency. 
    • Default A/P account. The default A/P account for transactions in this currency. 
  3. Click Write and close.
