This list is used to manage currencies used in the application. The required currencies can be picked from the Currency classifier (recommended) or entered manually.

The currency list is accessible from the Reference data section.

The list includes:

  • Numeric code - a unique numeric ID compliant with international standards. It is used to identify currencies when downloading exchange rate data from the Internet sources.
  • Alphabetic code - a brief currency name compliant with international standards.
  • Currency name - full name of the currency.
  • Rate - exchange rate for today.
  • Unit conversion factor - set to 1 by default. This parameter is not displayed. Unit conversion factors are used for currencies with extremely low exchange rates against US dollar (below 1:10 000). For example, the exchange rate of Viet Nam dong is 4.5826 USD per 100 000 VND. This field includes three subcolumns:
    • USD per - logical connective.
    • Unit conversion factor - exchange rate multiplier ratio.
    • Brief name - alphabetic code of the currency

Adding currencies from Currency classifier

  • Click Pick from classifier. This is a recommended method to add currencies, as it ensures correct currency data input.
  • The following fields wil be filled in automatically: Name, Numeric code, Alphabetic code. The Download from Internet check box will be selected automatically if exchange rates for this currency can be downloaded from online sources.

Adding currencies manually

  • Click Create.
  • Fill in the currency data.

Importing exchange rates

  • To start importing exchange rates from online sources, click Import currency rates. 
  • A predefined scheduled job named Exchange rate import is available in the application for automated download of exchange rates from online sources.
