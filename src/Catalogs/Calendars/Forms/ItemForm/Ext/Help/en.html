Information on work schedules

Create as many work schedules as necessary on the basis of the business calendar. Work schedules can apply to the whole organization or to a specific department (example: warehouse work schedule only applies to warehouse).

Depending on work specifics of the organization or a department, work schedules can be prepared for a calendar year or for a lesser period. If necessary, work schedules can be expanded on a regular basis during the year, each expansion covering as little as several days ahead.

Creating work schedules

  • Enter Description.
  • Specify Schedule filling period (for  example, January 01, 2014 - July 31, 2014). You can do this by:
    • typing in the dates manually.
    • selecting the dates from calendar (to access the calendar, click ).
    • by default, the period starts on January 1st of the current year. 
    • if the end of period is set to end of year, this field can be left blank.
  • Select Business calendar used as basis for the work schedule. If only a single business calendar is available in the organization, it will be selected automatically.
  • Select Filling method. This allows you to choose between two work schedule types available:
    • Weekly (by hour) - set Filling method to By week.
    • Cyclical (with user-defined cycle length) - set Filling method to By cycle.
    • enter the cycle length (example: 5 days).
    • enter the cycle start date (example: cycle starting from 01.01.2014).
  • Create Filling template. For this, you can choose any of two methods:
    • Select check boxes for workdays. This is sufficient to display both workdays and weekend days in the work schedule.
    • Specify work hours for each workday via the Fill in schedule hyperlink. Once this is complete, the hyperlink will change to display the total number of work hours and work mode.
      • Specify Work hours for each workday of a week (or cycle) by clicking Add. Enter Start time and End time. Any breaks in the daily work schedule are indicated by creating multiple entries (work hours before the break, and work hours after the break).
      • Only work hours are included in the schedule. Weekend days are not included.
  • If necessary, select the Exclude holidays check box. This will automatically exclude any dates specified as holidays in the selected business calendar from the work schedule.
  • If holidays are excluded from the work schedule and pre-holiday days are assigned a separate work schedule, click the Fill in schedule hyperlink to fill in Pre-holiday schedule (the procedure is identical to work schedule filling in).
  • Specify Planning horizon - if this is specified, the schedule filling period will be automatically compared to the number of months in the planning horizon, and a warning will be displayed if the schedule was not filled in to the required date.
  • Type in Description - enter detailed schedule information here. Filling in this field helps to navigate through extensive work schedule lists.

      Viewing work schedules

      Click Fill result. This will display all workdays and week-end days broken down by month. Click Fill setup to continue filling in the work schedule.

      Notes on filling in weekly work schedules

      Weekly (or hourly) work schedules are defined for each day of the week. This is the most basic and common type of work schedule. 40-hour workweek is a good example of this schedule type, as well as 5-day workweek, 2-day workweek, etc.

      To create a work schedule of this type, set Filling method to By week and specify Work hours for each workday of the week.

      Notes on filling in user-defined cyclical work schedules

      Cyclical work schedules are defined for specific dates; a date is selected as cycle day 0 and all other cycle days are numbered after it. Work schedules of this type are not affected by days of the week, and rarely affected by holidays.

      Cyclical work schedule examples are: shift-based schedules, and sliding week-end schedules (one workday, three days off; two workdays, two days off; etc)

      Let us review an example work schedule for a warehouse that works around the clock for 4 days, after which, the warehouse closes for 1 day for equipment maintenance and sanitation. In this case, the total cycle length is 5 days.

      • Set Filling method to By cycle.
      • Specify cycle length (5 days).
      • Specify cycle start date (03.01.2014. The cycle starts from the start date of the schedule filling period (01.01.2014), but you can change this if necessary.
      • Create Filling template and specify Work hours for each workday in the cycle.
