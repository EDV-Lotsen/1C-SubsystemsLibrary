World country classifier based on ISO 3166 standard and is intended for identification of countries of the world.

This classifier contains three structural parts:

  • digital identification,
  • titles,
  • alphabetic identification.


Digital identification part contains 3-digit digital codes of countries that is generated as a serial number.

Titles part includes a short and full official names of countries. If the full name is not set then full name matches the short name of the country.

Alphabetic identification part contains a 2-digit (alpha-2) and 3-digit (alpha-3) alphabetic codes that use latin letters. The main principle of those codes is that they in most cases have visual association with country names in English, French or other language.

Two-digit alphabetic codes are recommended for international implementations, they allow to create visual association with common used country name without any reference to its geo coordinates or state.

Three-digit alphabetic codes are used in specfic cases.
The digital code allows a system keep the same input codes if a country name or alpha codes were changed.
