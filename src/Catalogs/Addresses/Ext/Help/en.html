The Addresses/Contacts list stores addresses/contacts of a specific company. 

To add an address

  1. In the Lists section, click Companies.
  2. Double-click the company whose address you want to add.
  3. In the left pane, click Addresses/Contacts.
  4. Click Create, and then enter or select the following:
    • Address ID. The short address description. It is how the address is shown in the list of addresses.
    • First name, Middle name, Last name, Phone, Fax, E-mail, Address line 1, Address line 2, City, Region/State, Country, and ZIP code. The address information. 
    • Remit to. This name is used on printed checks written to this company, if different from the main company name. You only have to fill this field in the default billing address because this address is always used in checks.
    • Default billing. Indicates whether this address is the default billing address. A company must have a single default billing address.
    • Default shipping. Indicates whether this address is the default shipping address. A company must have a single default shipping address. When you select a company in a document, the default shipping address is selected in the Ship to field.
  5. Click Write and close.
