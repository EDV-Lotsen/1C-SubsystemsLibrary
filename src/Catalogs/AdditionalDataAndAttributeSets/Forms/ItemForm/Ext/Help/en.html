This form is used to view sets of custom fields and data.

It displays Description of the set and the catalog group corresponding to the set.

It also displays the lists of custom fields and data included in the set. The lists are read-only and cannot be modified in any way. To modify the list contents, use Custom fields or Custom data commands available in sections Administration - Common settings or Integrated subsystems (part 2).
