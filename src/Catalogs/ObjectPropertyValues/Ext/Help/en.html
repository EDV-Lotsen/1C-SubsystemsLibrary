This list is used to store values of custom fields (data).

You can work with custom fields (data) of multiple types. Data type is selected in settings of custom fields (data). When selecting Additional value data type, the user is also prompted to enter data values into the list. For each custom field or data of this type, a separate value list is used. This data type can be used to describe various properties of applied objects.

Additional value data type is commonly used; prompting users to select a value from the list expedites data input and reduces the input error rate.

Both standard and grouped value lists are used.

Selecting values

  • Select an item and click Select.
  • Or double-click the item.

Entering values

To enter values, click Create and fill in the required fields.

Grouping values

Extensive value lists can be grouped for convenience. To group a list, enter the desired number of groups. Groups can be independent or hierarchical.

  • To add a group to the list, click Create group and fill in the required fields. 
  • Drag the values to the corresponding groups.
