This form is used to enter values of additional attributes (data).

You can work with additional attributes (data) of multiple types. Data type is selected in settings of additional attributes (data). When selecting Additional value data type, the user is also prompted to enter data values. For each additional attribute or data of this type, a separate value list is used.

Entering values

  • Enter Description of an additional attribute (data) value.
  • Included in group field can be used to include the new value into an existing group. To create a new group, click Create (or press F8) and enter the group name. To select an existing group, click Show all (or press F4) and select a group from the list. 
  • If Use weight coefficients for values check box is selected for an additional attribute (data), enter numerical value in Weight coefficient field.
    • This field can be useful for analytical and reporting purposes. You are free to decide how you prefer to use weight coefficients in your analytics and reports. For example, weight coefficients for "Reliability" attribute can be managed in this way: set weight coefficients for attribute values 1-3 to zero; set weight coefficients for attribute values 4-5 to 1. This technique is useful for fast selection of reliable customers with positive weight coefficients.
