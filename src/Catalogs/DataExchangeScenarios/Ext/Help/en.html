This list is used to store data synchronization scenarios.

You can access the list from the Data synchronization section. Alternately, you can access the scenario list from the Data synchronization list via All actions menu.

The list includes:

  • Description - full scenario description;
  •  Use scheduled job - this icon indicates that scenario runs automatically on schedule.

You can add extra fields, such as Data version, Comment, and more, via the All actions - Customize form command.

Setting up scenarios

  • Click Create and fill in the required fields.

    Data synchronization scenario setup procedure is performed in the following steps:

    • Set a data synchronization scenario schedule.
    • Select infobases participating in the data synchronization scenario.
    • For each infobase, select a connection method and current action (sending or receiving data).
  • After the data synchronization scenario setup is complete, the data synchronization setup record in the list will be marked with . Data synchronization is performed on schedule, as a scheduled job.

Enabling scheduled execution

Click to Enable or Disable scheduled synchronization. If scheduled data synchronization is enabled, this line will be marked with a relevant icon.
