The Work Time Types form allows you to view the list of time types used for payroll calculation (regular work time, overtime, sick leaves, and so on), as well as create, edit, and delete them.

For each work time type, the following information is available:
  • Alphabetic code. The alphabetical code of the work time type.
  • Digital code. The numeric code of the work time type.
  • Description. The description of the work time type.
