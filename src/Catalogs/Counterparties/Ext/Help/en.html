The Companies list stores the companies you do business with - your customers and vendors. 

The system uses a unified list that stores both customers and vendors as opposed to two separate lists, which allows you to easily address the situation when a company is both a customer and a vendor. You can also create two separate records if a company is a customer and a vendor, if it better suits your situation. Companies are tagged as customers or vendors to expedite data entry—the sales documents in the Customer field show only companies that are tagged as customers, and the purchasing documents in the Vendor field show only companies that are tagged as vendors.

The "Our company" item (with a yellow dot next to its name in the list) represents your company. When you start working with AccountingSuite, update it with the actual details of your company: name, addresses, and so on.

To add a company

  1. In the Lists section, click Companies and then click Create.
  2. Enter or select the following:
    • Code. By default the system assigns consecutive codes when companies are saved, and no user input is required.

      You have the option to enter a custom code instead. Codes can include prefixes. For example, if you specify the code "C-001", your next companies are automatically numbered "C-002", "C-003", and so on.  
       
    • Name. The company name.
    • Customer and Vendor. Indicates whether the company is your customer or your vendor.

      You can select both options. The field names can be customized (for example, to Client and Supplier) in the Vendor name and Customer name fields in Accounting settings. 
       
    • 1099 vendor. Indicates whether the company is a 1099 vendor.

      This check box is only available if the Sales tax (US) check box is selected in Accounting settings. 
       
    • Tax ID. The taxpayer identification number of the company.

      This field is only available if the Sales tax (US) check box is selected in Accounting settings.
       
    • Default currency. The default currency for the transactions with this company.

      This field is only available if the Multicurrency check box is selected in Accounting settings.
       
    • Sales tax code. The sales tax code of the company selected from the Sales tax codes list.

      This field is only available if the Sales tax (US) check box is selected in Accounting settings. 
       
    • VAT reg. number. The VAT registration number of the company.

      This field is only available if the VAT check box is selected in Accounting settings. 
       
    • Terms. The default payment term for the transactions with this company.
    • Expense account. The default G/L expense account of the company.
    • Income account. The default G/L income account of the company.
    • Website. The company website.
    • Notes. Your notes.
  3. Click Write and close.
     

When you add a company, it will have a single placeholder address/contact record. Edit this record and add more addresses/contact records, if needed. A company must have one default billing address and one default shipping address.

To add a company address/contact

  1. In the Companies list, double-click the company.
  2. In the left pane, click Addresses/Contacts.
  3. Click Create, fill the address/contact fields, and then click Write and close.
