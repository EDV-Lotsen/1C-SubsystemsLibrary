The external user groups form is intended for viewing and editing group properties.

You can specify Participant type (for example, counterparties) of a user group. To implicitly include all external users of the specified participant type in a group, select the All check box.

When editing user groups, take into account that they are implicitly included into the parent group content. To add a large number of users you can use the Fill command.
