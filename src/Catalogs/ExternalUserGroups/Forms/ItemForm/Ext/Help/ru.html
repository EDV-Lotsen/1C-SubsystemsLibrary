Для группы внешних пользователей задается список входящих в нее внешних пользователей.
При этом нужно учитывать, что внешние пользователи группы неявно входят в состав групп-родителей.

Для заполнения в группу большого числа пользователей применяйте кнопку Подбор.

Подробнее см. справка по группам внешних пользователей.
