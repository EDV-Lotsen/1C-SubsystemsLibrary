This list contains users of the application.

In case of large number of users this list can be grouped. For example by a department of company where the user is working. To see all users of subordinate groups for current group you can set Show subordinate group users check box in the bottom of the list. To add new user groups you can use commands on the left of the list. To add a bulk of users to a group you can use Fill command in form of group. One user can be included in one and more user groups.

To manage users you should have administration right. If you do not have this right, you are allowed only to view your user data and change your password, application language, and contact information.

Users can be marked as inactive. To view these users you can use Show inactive users command in All actions menu of the command bar of user list form.
