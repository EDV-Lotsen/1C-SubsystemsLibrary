The user card allows seeing and editing features of informational base user. Particularly, here could be adjusted parameters of authentication (appearance of the form repeats the editing form of authentication parameters of informational base user in Configurator), assign the roles and edit the user’s contact information. 

Usually users are connected to the relevant users of informational base. To add the connection with user of informational base it’s necessary to set the flag “Access to informational base is allowed”. To delete the access you have to unset the flag.

To edit the features of informational base user it’s necessary to have administrator’s rights and the right to add and change the user. It’s possible to change some private features. To assign roles (if the direct role assignment is used) you must have administrative rights.
