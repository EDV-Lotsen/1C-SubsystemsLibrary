Назначение формы "Users информационной базы".

  • Найти некорректных пользователей информационной базы и выполнить исправления:
    • либо пользователь был добавлен с помощью конфигуратора (или программно нестандартно):
      • можно удалить пользователя,
      • можно создать связанного с ним пользователя в справочнике;
    • либо пользователь был удален с помощью конфигуратора (или программно нестандартно):
      • можно повторно разрешить доступ к информационной базе;
    • либо полное имя пользователя было изменено с помощью конфигуратора (или программно нестандартно):
      • можно открыть и записать пользователя,
        в результате чего полное имя будет перезаписано.
  • Найти пользоватей и внешних пользователей без пользователя информационной базы
    (когда доступ к информационной базе не разрешен),
    например, пользователь был уволен и оставлен в справочнике для истории.
  • Посмотреть список всех пользователей информационной базы.
