The purpose of form "Users of information base".

  • Find invalid user of informational base and perform corrections:
    • if the user was added with the help of configurator (or nonstandard by a software):
      • you can delete the user,
      • you can create the user, linked to invalid user from the catalog;
    • if user was deleted by the designer (or nonstandard by a software):
      • you can re-enable access to the informational base;
    • If the full name of the user was changed by the designer (or nonstandard by a software):
      • You can open and store the user,
        as the result the full name will be rewritten.
  • Find users and external users without the informational base user
    (when access to the informational base is forbidden),
    For example user was fired and left in the catalog for history.
  • See the list of all informational base users.
