Edit additional settings of email account. These settings allow to set additional parameters of email account, such as: authentication, server ports, idle time, and and other.

  • Common
    • POP3 authentication - configuring authentication for POP3 server (incoming mail server);
  • Outgoing mail server - authentication settings for SMTP server
    • SMTP server requires authentication - if this check box is set then SMTP server requires authentication then it will be possible to select one of authentication methods:
      • Same as incoming mail server - user and password are used from incoming mail server parameters, SMTP authentication will be set to "default";
      • Set authentication parameters - authentication parameters are set manually:
        • SMTP user;
        • SMTP password;
        • Authentication type for SMTP server.
      • Log on to incoming mail server before sending.
  • Advanced - POP3 and SMTP server ports, wait for successful completion timeouts, and number of days to keep emails on server
    • Setup server ports / POP3 server - POP3 server port;
    • Setup server ports / SMTP server - SMTP server port;
    • Server wait timeout - the time in seconds to wait for successful completion of an action;
    • Keep message copies at server - wefines whether to keep emails on server;
      • Keep email copies <days> - the number of days during which emails will be kept on server.
        Emails with expired storage period will be removed during next time get new emails action will be performed and they will not be transferred as incoming messages.
