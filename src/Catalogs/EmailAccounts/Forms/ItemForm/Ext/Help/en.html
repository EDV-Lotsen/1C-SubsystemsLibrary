User info:

  • Name - the name owner accounting records, for example Ivan S.
  • Email - email address in the following format: user@mailserver

    • Server properties:

  • POP3 server server - POP3 server address (incoming mail server)
  • SMTP server server - SMTP server address (outgoing mail server)

    • Authentication:

  • User - the user name of email account (login)
  • Password - the password of email account
  • Remember password - if this check box is set then the password will saved in the application and user will not be prompted for password in future*.

    • Use Check account command to verify email account parameters. This command sends a standard email to checks that connection to incoming mail server can be established if this email account is used to receive emails.

    Email account is used only for sending emails check box can be set only system accounts that are used for sending emails.

    * - We recommend that you to select the Remember password check box for accounts that are used for system purposes and must operate without of user attention. But it is still up to you whether to store passwords in the application or not.
