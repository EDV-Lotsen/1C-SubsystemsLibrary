Card of an external user allows viewing and editing user’s features of informational base. Particularly here you can adjust parameters of authentication (the appearance of form repeats the editing form of authentication parameters of informational user base in the Configurator), aassign roles (if a direct role assignment is used).

Typically, the external users are connected to relevant users of information base. To add the connection with the user of informational base you have to set flag “Access to informational base is allowed”. For deletion you have to unset it.

The external users are always connected to the informational base object, if the connection is set, it can’t be changed.

To edit features of external user of informational base you need administrator’s rights or the right to add and change external user. It’s possibly to change some features. To assign roles (if a direct role assignment is used) you must have administrative rights.
