В справочнике Внешние пользователи ведутся те пользователи, кому предоставлен вход в ИБ «извне». Их доступ, как правило, сильно ограничен, они не принимают активного участия в процессах учета, работают обособленно, не взаимодействуют между собой.

Справочник предназначен для ведения списка внешних пользователей системы. Как правило, элементы справочника связаны с соответствующими пользователями информационной базы. С помощью команды "Users информационной базы" администратор системы может выполнить анализ соответствия элементов справочника Внешние пользователи с пользователями информационной базы.

Основной способ добавления и редактирования элементов справочника через команду Внешний доступ, которая размещается в командной панели формы объектов информационной базы, например, контрагенты, физические лица. При выполнении команды сразу создается элемент справочника и связывается с объектом информационной базы или открывается существующий элемент.

Внешних пользователей в справочнике можно группировать, например, по видам отношений с ними: партнеры, контрагенты и т.д. Для этого в левой части списка необходимо создать необходимое количество групп внешних пользователей и добавить в них внешних пользователей.

Внешнему пользователю без административных прав доступны для просмотра только сведения о себе. Он также имеет права на изменение некоторых свойства таких как: имя, пароль, язык системы и контактная информация.

Редактор внешних пользователей может использовать команду Внешний доступ, редактировать справочники Внешние пользователи и Группы внешних пользователей, но не может редактировать права доступа непосредственно (а только через изменение состава групп внешних пользователей).
