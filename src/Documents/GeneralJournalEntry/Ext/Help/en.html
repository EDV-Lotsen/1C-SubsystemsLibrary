The general journal entry is a document used to create accounting transactions.

The system does not allow saving unbalanced transactions unless the Unbalanced G/J entry posting check box is selected in Accounting settings. It is recommended that you only allow posting unbalanced transactions when entering beginning balances.

To enter a general journal entry

  1. In the Accounting section, click General journal entries, and then click Create.
  2. Enter or select the following:
    • Number. By default the system assigns consecutive numbers when documents are saved, and no user input is required.

      You have the option to enter a custom number instead. Document numbers can include prefixes. For example, if you specify the number "E0001", your next entries are automatically numbered "E0002", "E0003", and so on.
    • Currency. The transaction currency.

      This field is only available if the Multicurrency check box is selected in Accounting settings.
    • Date. The entry date.

      This field is prefilled with the current date.
    • Exchange rate (shown as currency symbols, for example, $/1€). The exchange rate between the transaction currency and the reporting currency.

      The system retrieves the exchange rate from the Exchange rates list.

      If you change the entry date, the system recalculates the exchange rate. If you change the date or currency, the system does not recalculate prices in the line items, therefore it is recommended that you specify the date and currency before entering the line items.
  3. In the line items, add accounting transactions.
  4. In the Memo field, you can enter notes on the entry.
  5. Click Post and close.
