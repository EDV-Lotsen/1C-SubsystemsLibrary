This form is used to add digital signatures to files. This could be useful, for example, when the signature file was obtained after the signed file.

To open this form, execute the DS and encryption - Add DS from file command located in file list (list of attached object files), the Attaches files list, or file card (attached file card). This command can also be located in the All actions menu.

The list includes:

  • DS file - path to the DS file.
  • Comment - additional information on the DS file.

Add digital signature from file

Select one or several files containing digital signatures.

  • Click  Add,  then click  to specify path to the DS file located on a local or network hard disk.
  • Enter Comment for the signature.

Deleting digital signatures

  • Click to Delete a DS file from list.
