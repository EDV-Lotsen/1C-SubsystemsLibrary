This list is used to select exchange plan nodes (infobases).

To access the list from another applied object, use Pick or Add command; for example, when entering Data import prohibition dates.

The list includes:

  • Application - list of application with data synchronization enabled. For each application, infobases are grouped by data synchronization type - for example, infobases not using exchange rules.
  • Infobase - list of infobases with data synchronization enabled. Keep in mind that you can specify <All infobases> value for any application or any data synchronization type - this can be useful if you need to select all infobases of specific version and specific data synchronization type.

Selecting exchange plan node

Select a value and click Select.

Selecting multiple exchange plan nodes

Select multiple values and click Select.
