This form is used to specify path to the initial image file archive (in case of volume-based file storage).

If the file storage is based on disk volumes, an archive containing the volume-stored files will be automatically created together with the initial image.

When creating a subordinate node infobase from the initial image in the target application, you will need to get the files from this archive. For this, execute the Store initial image files command located on actions panel of the Organizer section. All further data synchronization iterations will use the volume-based files automatically.
