This list is used to select digital signature certificates (DS) to be used during file encryption.

To open the list, execute the DS and encryption - Encrypt command located in the file list (list of attached object files), attached file list, or file card (attached file card). This command can also be located in the All actions menu.

The list includes:

  • Selected - this is a check box;
  • Issued to - owner of the digital signature certificate;
  • Issued by - name of the DS issuer organization;
  • Expires on - DS certificate expiration date;
  • Purpose - DS certificate usage goal, for example, Secure email exchange, Server authentication, etc. To view detailed information on the certificate purpose, hover mouse pointer over the corresponding cell.

Encrypting files

  • You can use multiple certificates during file encryption. For example, it may be necessary when encrypting a file for several users.
  • Prior to file encryption, select the check boxes for the corresponding users' certificates.
  • The lines containing the selected sertificates will become disabled. When encrypting all listed files, clearing the selected check boxes will not be possible.
  • Click Encrypt to confirm.
  • If a file has multiple versions, all versions will be encrypted. This may take a significant time.
    • In this case, a warning message will be displayed before the encryption procedure starts. Click Yes to confirm.  
  • The certificates used for encryption will be listed for viewing on the Encrypted for tab of the file card. The Encrypt command will be disabled in this case.
