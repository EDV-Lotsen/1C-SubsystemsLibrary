This form is used to enter password during file decryption procedure.

To open this form, execute the DS and encryption - Decrypt (All actions - DS and encryption - Decrypt) command located in the file list (attached file list), attached file list, or file card (attached file card).

The following data is displayed:

  • File - this field displays the file name. It is not editable.
  • File encrypted for - this field contains the list of personal digital signature certificates (DS) for which the file is encrypted. It is not editable.

    Decrypting files

    • Enter the file decryption password into the Password field and click OK.
