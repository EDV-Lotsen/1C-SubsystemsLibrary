This list is used to view and enter values of additional data. Generally, values of additional data are entered in a separate window, as opposed to values of additional attributes.

The list is accessible from applied object windows with additional data set up via the Additional data command.

The list includes:

  • Description - name of additional data field;
  • Value - field used for viewing or entering values of additional data.

Entering values

  • To switch to additional data value input/edit mode, double click the corresponding field.
  • Value input methods may vary depending on the additional data type (for more details on types of additional attributes and data, see here).
  • Additional data may have only two allowed values, such as Yes and No. In this case, new values cannot be added.
  • If a list of values is available for input, click Show all (or press F4). Then select a value from the list. Depending on the settings, value lists can be either hierarchical or nonhierarchical. New values can be added to these lists if necessary.
  • Depending on the settings, values may also be selectable from the application catalogs.
  • Depending on the additional data settings, you may be required to  Select data type.

Changing additional data content

Users with sufficient rights can create new additional data or modify settings for existing additional data. For this, execute All actions (or More, in Taxi interface) - Change additional data content command.

Deleting additional data

Additional data marked for deletion is identified by icon. Before additional data can be deleted, it may be necessary to  Clear values. Double click a value to switch to edit mode.
