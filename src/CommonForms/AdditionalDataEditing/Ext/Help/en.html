This list is used to view and enter values of custom data. Generally, values of custom data are entered in a separate window, as opposed to values of custom fields.

The list is accessible from applied object windows with custom data set up via the Custom data command.

The list includes:

  • Description - name of custom data field;
  • Value - field used for viewing or entering values of custom data.

Entering values

  • To switch to custom data value input/edit mode, double click the corresponding field.
  • Value input methods may vary depending on the custom data type (for more details on types of custom fields and data, see here).
  • Custom data may have only two allowed values, such as Yes and No. In this case, new values cannot be added.
  • If a list of values is available for input, click Show all (or press F4). Then select a value from the list. Depending on the settings, value lists can be either hierarchical or nonhierarchical. New values can be added to these lists if necessary.
  • Depending on the settings, values may also be selectable from the application catalogs.
  • Depending on the custom data settings, you may be required to  Select data type.

Changing custom data content

Users with sufficient rights can create new custom data or modify settings for existing custom data. For this, execute All actions (or More, in Taxi interface) - Change custom data content command.

Deleting custom data

Additional data marked for deletion is identified by icon. Before custom data can be deleted, it may be necessary to  Clear values. Double click a value to switch to edit mode.
