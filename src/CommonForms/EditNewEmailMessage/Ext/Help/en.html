You can specify several email message recipients in the Address recipient field.

You can specify the email address where you want recipient to reply in the Reply to field. Fill this field if you do not check the system email account for incoming correspondence (for example, if the email account is used only for sending messages).

Use the Attach file button to attach files. To delete the attachment, place the cursor on it and press Del.

Email address examples:

  • ivanov@server - brief record.
  • Ivanov ivanov@server - detailed record that includes both recipient name and email address.
  • ivanov - to write the recipient in this way, make sure that the email server can match brief name with the full address.
