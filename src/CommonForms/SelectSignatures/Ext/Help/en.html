This list is used to select digital signatures to be saved with a file.

To open this list, execute the DS and encryption - Save with DS command located in the file list (list of attached object files), attached file list, or file card (attached file card), provided that the Actions on saving with DS custom setting is set to Ask. This command can also be located in the All actions menu.

The list includes:

  • Save - this is a check box.
  • Certificate owner - user that puts their signature on the file;
  • Signature date;
  • Comment - detailed information on the digital signature.

Select check boxes for digital signatures to be saved with the file and click OK.

To disable this window and always save all digital signatures automatically, select the Don't ask again check box.
