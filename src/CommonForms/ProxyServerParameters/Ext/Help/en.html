To access the Internet from client computers, the proxy server parameters must be specified on each client workstation (via Personal settings of the Administration section).

When 1C:Enterprise runs in the client/server mode, the proxy server parameters must also be specified for the 1C:Enterprise server (via the Setup and administration - Common settings section). Proxy server parameters for a client workstation and for a 1C:Enterprise server are not necessarily identical.

Important: if security profiles are enabled, you must use the inetcfg.xml configuration file to set up the proxy server parameters.
