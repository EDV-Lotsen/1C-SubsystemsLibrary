To access the Internet from client computers, the proxy server parameters must be specified on each client workstation (with Personal settings of the Setup and administration section). When 1C:Enterprise runs in the client/server mode, the proxy server parameters must also be specified for the 1C:Enterprise server (with the system parameter settings form of the Setup and administration section). In the general case, the proxy server parameters for a client workstation and for a  1C:Enterprise server can be different.
