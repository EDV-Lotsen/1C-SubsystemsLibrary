This form is used to save print forms. Print forms can be saved locally in user-selectable format, or attached to documents.

  • Use the radio button to select how you want to save the print form:
    • Save to local directory.
      •   Specify path to the local directory.
    • Attach to document.
  • Prior to saving or attaching a print form, you need to specify one or more formats for it via check boxes.
  • To compress the print forms before saving, select the Archive attachments (.zip) check box.
  • Click Save. Your selection wil be saved.
