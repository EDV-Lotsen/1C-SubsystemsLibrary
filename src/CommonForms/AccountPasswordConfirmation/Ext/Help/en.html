For sending the email letter, type the password and click the Continue command panel button. This form is opened when one is sending a letter using an account with no password set.

The form command panel contains the following buttons:

  • Continue - closes the form and sends the letter.
  • Cancel - cancels sending the letter.
  •  - shows this help.

 

If Store password is selected, the password is stored in the infobase system settings.
