The Accounting settings form allows you to specify accounting settings.

To specify accounting settings

  1. On the main menu, click Desktop.
  2. In the bottom part of the window, click the Accounting settings button.
  3. Specify the settings on the Main tab.
    • Display company codes. Adds the Company code field to documents and lists.
    • Beg. balances in documents. Adds the Beginning balance check box in documents.

      This allows using invoice and return documents for entering beginning balances. It is recommended that you clear this check box after you enter the beginning balances.
       
    • Multilocation. Adds the multilocation functionality, including:
      • Locations list in Warehouse
      • Warehouse transfer document in Warehouse
      • Location field in documents
         
    • Multicurrency. Adds the multicurrency functionality, including:
      • Currencies list in Lists
      • Exchange rates list in Lists
      • Currency and exchange rate fields in documents
         
    • Unbalanced G/J entry posting. Allows posting unbalanced general journal entries. 

      This allows using general journal entries for entering beginning balances. It is recommended that you clear this check box after you enter the beginning balances.
       
    • Default currency. The reporting currency.
       
    • Disable LIFO (IFRS). Prohibits saving new inventory items with the LIFO costing method.

      LIFO costing method is prohibited by IFRS.
       
    • Vendor name. The vendor name in document forms, for example "Supplier".

      The default name is "Vendor".
       
    • Customer name. The customer name in document forms, for example "Patient".

      The default name is "Customer".
       
  4. Specify the settings on the Financial localization tab.
    • Sales tax (US). Adds the US financial localization functionality, including:
      • Sales tax codes list in Sales
      • 1099 categories list in Accounting
      • Sales tax type field in document line items
      • Taxable amount field in document line items
      • 1099 category field in Chart of accounts 
    • VAT. Adds the VAT financial localization functionality, including:
      • VAT codes list in Accounting
      • VAT tab in this form
      • VAT-related fields in document line items
         
  5. If you selected VAT in the previous step, specify the settings on the VAT tab.
    • Default purchase VAT. The default purchase VAT.

      When you create a new item, it prepopulates the Purchase VAT code field.
       
    • Default sales VAT. The default sales VAT.

      When you create a new item, it prepopulates the Sales VAT code field.
       
    • Price includes VAT. The default value of the Price includes VAT check box in documents.
       
  6. On the Posting accounts tab, specify the default accounts used to automatically create transactions when you post documents.

    Detailed information on accounts used in automatic G/L posting is provided in the General journal help section.
