On this form you can specify the following server cluster access parameters for managing the client/server infobase:

  • infobase administrator name and password for locking scheduled and background jobs.
  • server cluster administrator name and password on behalf of which user session termination is executed. Leave this field empty if no administrators are specified in the server cluster.
  • port numbers of the server cluster and server agent. Leave this field empty if the cluster and agent have standard ports.
