This form is used to set up or validate inter-application connection parameters used for data synchronization.
Each tab displays parameters for a specific connection method.

The connection parameters are also displayed during data synchronization setup procedure in the Data synchronization creation wizard. The default parameters are displayed; they can be modified if necessary.

To start setting up Connection parameters, use the related command in the data synchronization settings.

The following parameters are common for all connection methods:

  • Default connection - select a connection to be used as default during both automated and interactive data synchronization.
  • Test connection - click to validate the connection parameters and attempt connecting with these parameters.
  • Compress sent data - select this check box to reduce the size of sent messages. For direct infobase connections, this check box is disabled.
    • Exchange message archive password - enter the password required to compress or decompress exchange messages. For successful data exchange, compression password in the source application and decompression password in the target application must be identical.

Direct connection:

  • Use the radio button to select an infobase type:
    • Correspondent file mode infobase
      •  Select Infobase directory.
    • Correspondent client/server mode infobase
    • Enter Server name;
    • Enter Server infobase name.
  • Select the OS authentication check box if necessary (this check box is cleared by default). In this case, the following fields should remain blank.
  • Enter User name;
  • Enter User password.

      Internet

      • Enter application URL.
      • Enter User name and Password.
      • Select the Remember password check box if necessary.
      • Click the Internet access parameters hyperlink and configure proxy server parameters.

      Network directory

      In the Directory field,  Specify path to a local or network directory.

      FTP resource

      • Specify Directory path and data transmission Port.
      • Enter User name (login) and Password.
      • Enter Maximum file size if the selected FTP resource has limited storage.
      • Select the Passive connection check box if necessary.
      • Click the Internet access parameters hyperlink and configure proxy server parameters.

      Email messages

      In the Email account field, select an email account from list or create a new account.

      Enter Maximum email attachment size if necessary.
