The form is intended for setting up and checking exchange message transports.
If a new record from an information register list form is created, the Infobase node field is available.
In the Infobase node field, specify the exchange plan node whose transport parameters are setting up.
Switch between the form tabs to set up specific transport kinds.

The following settings are common for all transport kinds:

  • "Default exchange message transport kind" –  transport kind to be used by default for automatic and interactive data exchange.
  • "Exchange message archive password" – password for unpacking exchange message archives. Passwords must be the same in both infobases that take part in the exchange.
  • "Exchange log file name" – If the exchange protocol file has been set, all errors that occur during the data exchange are written to this file.
  • "Execute data exchange in debug mode" – flag that shows whether the data exchange is executed in the debug mode. In the debug mode the exchange will not be stopped if an error occurs. The exchange with errors will be completed and debug messages will be written to the exchange log and event log. Use this mode to debug exchange rules.
