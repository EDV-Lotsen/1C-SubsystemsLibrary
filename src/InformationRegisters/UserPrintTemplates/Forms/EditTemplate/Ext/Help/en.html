Templates in spreadsheet document format can be edited directly in the application. Templates in office document formats (Microsoft Word, OpenOffice.org Writer) are edited in the corresponding software, which needs to be installed separately.

You do not need to launch the editing software manually. Simply click a template file to start editing it; the corresponding editor will launch automatically (provided that it is installed on your computer).

Web client may require additional setup before you can start using it to edit templates.

If corresponding editor software is already installed on your computer, click Continue.

Each template type (format) requires a specific editor software:
