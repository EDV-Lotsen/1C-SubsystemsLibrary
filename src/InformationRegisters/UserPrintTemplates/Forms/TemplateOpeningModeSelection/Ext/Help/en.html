This form is used to select template opening modes.

To open a template in the Print form templates list:

  • press Enter and select a template opening mode;
  • double-click the template and select a template opening mode;
  • click  Open;
  • click  Change.

Template opening modes:

Use radio button to select the template opening mode you want to use when opening templates by double-clicking (or by pressing Enter):

  • Read-only - the selected template will be opened in read-only mode in an external viewer application; it will not be locked for editing.
  • Read/write - the template will be opened in read/write mode in an external editor; it will be locked for editing until you close it.

Disabling the message

  • To disable this message, click the Don't ask again check box.
  • To enable the message again, execute the All actions - Set action on print form template choice command located in the print form template list, or click the Set action on print form template choice button located on the Print page of the personal settings.
