This list is used to store and view currency exchange rates.

Currency rates can be automatically downloaded from the online sources via a scheduled job, or updated manually (see Exchange rate import).

Manual input is used whenever no online currency data sources are available.

The list includes:

  • Rate date;
  • Currency - alphabetic code of a currency compliant with international standards.
  • Rate;
  • Unit conversion factor.

Entering currency rates manually

  • Click Create, 
  • select a Currency from list (similar to the Currencies list).
  • enter Rate date, or select it from calendar, 
  • enter Rate,
  • enter Unit conversion factor (the default value is 1). Unit conversion factors are used for currencies with extremely low exchange rates against US dollar (below 1:10 000). For example, the exchange rate of Viet Nam dong is 4.5826 USD per 100 000 VND. In this case, you need to set Rate to 4.5826 and Unit conversion factor to 100 000.
