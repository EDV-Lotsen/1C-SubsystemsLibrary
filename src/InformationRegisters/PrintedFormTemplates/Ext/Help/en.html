The Print form templates form allows you to view print templates for various document types, as well as edit or delete templates.

To view a template

  • Select the template and click View.
     

To lock a template and open it for editing

  • Select the template and click Lock and Edit.
     

To save a template being edited

  • Close the template and confirm that you want to save it.
     

To unlock a template

  • Select the template and click Save and Unlock.
