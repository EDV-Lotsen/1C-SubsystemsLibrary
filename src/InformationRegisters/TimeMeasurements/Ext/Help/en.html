This list is used to view time measurements performed during application performance tests.

To access the list, execute the Time measurements command located on the navigation panel of a key operation.

The list includes:

  • Key operation - full name of a key operation;
  • Measurement start date - measurement start date and time;
  • Session number;
  • Execution time in seconds;
  • Record date - record date and time;
  • End date - measurement end date and time;
  • User - full user name
  • Local record date.

Click All actions - Customize form to add more fields. Examples:

  • Name - key operation name used in the application;
  • Priority - copied from the key operation list;
  • Minimum level - copied from the key operation list; 
  • Target time - copied from the key operation list.

Measurement data are saved automatically as key operations are executed. However, measurement data can be entered manually if necessary. This functionality can be useful for debugging purposes, among others. Click Create and fill in the required fields.
