To replace standard object conversion and object change record rules with rules from file you must:

  • Set a switch to Rules from file, click Import rules button, and select a file on a computer or in network directory;
  • Check that correct and suitable for source and target configuration version rules are imported;
  • Click Write and close button to apply imported rules.

Note: object conversion rules and object change record rules are applied for data exchanges of the same type (that are based on the same exchange plan; for example: there are two data exchanges Exchange with Miami branch and Exchange with Brazil branch based on Full exchange plan). Thus is why it is enough to replace exchange rules for any of data exchanges of the same type (you can select it from created data exchanges).
