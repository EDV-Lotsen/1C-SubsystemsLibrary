This list includes the latest information on users assigned to perform tasks on given task addressing object.

Roles with no assigned users are marked with the Attention icon. This can likely result in unassigned tasks (tasks that can not progress because no users were assigned to perform them). The Unassigned tasks report is used to monitor any unassigned tasks. The Task control manager predefined role can be used to send unassigned task notifications.
