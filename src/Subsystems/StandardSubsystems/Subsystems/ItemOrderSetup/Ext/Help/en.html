For a number of objects, order of items in lists can be specified. Use the Up and Down list form buttons for it. In hierarchical and subordinate lists, items can be move only in the limits of a group.

In the following cases an item position cannot be changed:

  • if the list is filtered;
  • if the list is ordered by any other attribute, for example, by name.

In such cases, the warning is displayed when you click Up or Down.
