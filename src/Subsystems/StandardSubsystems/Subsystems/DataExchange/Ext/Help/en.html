This application supports performing the data synchronization between infobases by data exchange rules.

The exchange rules are created in the Data conversion 2.1 application.

Data synchronization can be monodirectional or bidirectional. It can be performed automatically by schedule or manually by user request. Data synchronization can utilize a variety of communication channels: network folders, email, or FTP.

To set up data synchronization, use the Data synchronization command located on navigation panel in the Administration section.
