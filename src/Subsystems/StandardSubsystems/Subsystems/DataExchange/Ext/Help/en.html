The configuration is intended for performing the data exchange between various infobases using exchange rules.

Exchange rules are created with the "Data conversion 2.1" configuration.

The data exchange can be monodirectional or bidirectional. It can be executed automatically (by specified schedule) or manually (by user request). The data exchange can be executed through various communication channels: network directory, email, or FTP.

Use the Data exchange navigation panel command of the Setup and administration section to set up the data exchange.
