Print forms are available for many application documents. All print forms are governed by a unified mechanism that allows print form template editing without making any changes to the application itself. Print forms of the applied objects can be generated on the basis of spreadsheet templates (MXL) or office document templates (Microsoft Word, OpenOffice.org Writer).

Command bar of each application document supporting print form generation contains the Print command that can be used to generate print forms in a variety of formats. Document list forms contain similar commands. You can select and print multiple list documents with a single command.

Use this command to generate and display a print form in the dedicated Print document form. To print a generated print form, use the Print command.

  • Send by email - sends the generated spreadsheet documents as email attachments.
  • Go to document - scrolls the print form list until the print form of the selected document is displayed (provided that it was not displayed at the moment).

To access print form settings, execute the Print form templates command located in the Setup and administration section.
