The system is intended for managing users and external users. It synchronizes them with the infobase user list (to view the list, run the application in the Designer mode). This provides you with access to user details, as well as with the option to reference to users from other infobase objects.

 

Important: we recommend that you account infobase users and set up user rights in the "1C:Enterprise" mode only. In the "Designer" mode, it is allowed to create only users with full rights (system administrators). When users with full rights  start the infobase in the "1C:Enterprise" mode, correspondent items is automatically generated in the "Users" catalog.
