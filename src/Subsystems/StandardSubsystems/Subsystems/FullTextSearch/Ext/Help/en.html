This application allows to perform full-text search, looking for a specified string in any object attributes of any application object. If any attached files containing text data are found, their contents will also be searched.

The full-text search feature supports: latinization (conversion of non-Latin characters to Latin), substitution (using Latin characters instead of identically-looking non-Latin letters), and fuzzy search (finding strings approximately matching the search string).

To enable and set up the full-text search feature, use the system parameters setup form located in the Setup and administration section.
