This application is used to maintain infobase object change history. Version history provides quick and clear answers to questions like "Who changed this object", "When was this object changed last", or "What was changed exactly". Versioning can apply to catalogs and documents. The application allows to view any version of given object, or to identify differences between two versions of an object.

Before you can start working with object versioning functionality, you need to select the Use object versioning check box located in the system parameter settings form, in the Setup and administration section. This will make the Versioned objects command available on the navigation panel. Execute this command to proceed to the versioning settings form.
