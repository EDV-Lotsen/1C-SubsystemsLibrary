When you post a document that changes inventory balances, AccountingSuite automatically generates inventory journal entries based on the transactions listed in the document.

You cannot edit or remove entries that are generated automatically. However, you can use the Inventory beginning balance document for adding entries that are not based on documents, or for correcting inventory journal data. 

Each inventory journal entry includes the following columns:

  • Period. The day when the journal entry was created.
  • Recorder. The document that generated the journal entry.
  • Line number. The number of the line item that generated the journal entry.
  • Item. The item code.
  • Location. The warehouse location.
  • Layer. A document that received the inventory batch (purchase invoice or sales return).
  • Qty. The number of items.
  • Amount. The transaction amount calculated according to the costing method selected in Accounting Settings.
     

The following documents update the inventory journal:

  • Cash purchase
  • Cash sale
  • Purchase invoice
  • Purchase return
  • Sales invoice
  • Sales return
