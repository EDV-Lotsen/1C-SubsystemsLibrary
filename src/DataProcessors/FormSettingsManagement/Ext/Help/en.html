Users can customize the content and appearance of forms. Use the Change form form command for it. System administrator can copy form settings from one user to others with the Form settings management form (for example, to customize forms for a new user) and delete custom settings of the specified forms.

The list contains titles of forms that have custom field for users specified in the User field. If the user has no object form custom settings, the list is empty. To find a form in the list, type a part of form name in the Search string field and then click Search.

You can copy user settings to another user by clicking Copy to another user. If the user has custom settings for the selected forms all this settings will be replaced with new ones.

You can delete custom settings of the selected forms by clicking Delete settings. These forms will be opened with the default settings.
