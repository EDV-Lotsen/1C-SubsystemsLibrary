Full-text data search allows to search through contents of both card fields and attached files.

You can enter data URL in the search input field. This will open the related data card instead of performing data search. To get URL for an infobase object, use command  located in the header of each 1C:Enterprise 8 window.

In file mode Infobase, full-text search index is not updated automatically. You need to refresh it manually on a regular basis. If the full-text search index is not relevant, ie there is data that is not yet indexed, in file mode the Refresh index button becomes available.

Search example Result
Sales 00016422 Returns Goods and services sales document numbered 00016422.
Order 120000 Returns transfer orders for cash amounts 120000 and 120000.50
Johnson Acacia Returns all documents prepared by Mr. Johnson for Acacia Ltd. counterparty
Sales Acacia Returns all Goods and services sales documents related to Acacia Ltd. counterparty

Recommendations:

  • If your search returns too many results, refine your search by adding more data to the query (dates, amounts, counterparty names, etc). 
  • If your search returns too few results, expand your search by removing some data from the query
  • To perform exact search, use:
  • Quotes to search for an exact phrase. Example: "mini vacuum cleaner"
  • OR to search for any one of two words. Example: Acacia OR Uniconstructors search returns all references to Acacia Ltd. or Uniconstructors Inc. counterparties.
  • Brackets to group search terms. Example: Johnson (Acacia OR Uniconstructors) search returns all documents prepared by Mr. Johnson for Acacia Ltd. or Uniconstructors Inc. counterparty.
  • Use exclamation point (!) to search for synonyms. Example: !amount search returns all documents containing words amount or number.
  • Use hash sign (#) for fuzzy search. Example: Uniconstruct#3 search returns all documents containing words that differ from Uniconstructo by 3 or less letters
  • Use asterisk (*) to search for partial word. Example: Unico* search returns all documents containing words that start with Unico: Uniconstructors, Unicorn, etc.
