Currency exchange rates are downloaded from the Internet sources.

The exchange rate download window is divided into two sections:

  • Period: 
    • From - start date of the exchange rate download period.
    • To - end date of the exchange rate download period.
  • Currency list - by default, contains currencies from the Currencies list marked via the Download from Internet check box. The list includes:
    • Download - select this check box to download exchange rates for the specified period.
    • Alphabetic code - a brief currency name compliant with international standards.
    • Currency - full name of the currency.
    • Rate date - effective date of the downloaded exchange rate.
    • Rate - exchange rate for the specified date.
    • Unit conversion factor - used for currencies with extremely low exchange rates. The default value of 1 is not displayed.
  • to add fields to the list, execute the All actions - Customize form command:
  • Numeric code - a unique numeric ID compliant with international standards. It is used to identify currencies when downloading exchange rate data from the Internet sources.
