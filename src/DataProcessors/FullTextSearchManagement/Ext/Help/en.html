This data processor is used for full-text search index management. You can update or clear the index using this data processor.
In client/server mode, full-text search index is updated automatically.

Commands:

  • Update index
    		•  Updates the full-text search index by incorporating all objects that were created or modified recently.
  • Clear index
    		•  Deletes all full-text search index files.

     

    See also: Full-text search index, General information on forms.
