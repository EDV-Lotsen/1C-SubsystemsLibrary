The interactive data synchronization wizard is used to perform data synchronization with optional data preview and data mapping. This wizard allows to perform object mapping between two infobases participating in data synchronization. Once data mapping is complete, data import will be performed, followed by data export.

The interactive data synchronization mode should be used when identical objects were independently entered into both infobases participating in data synchronization. In other words, the interactive data synchronization mode should be used during the first data synchronization between two infobases previously used for accounting. When synchronizing such infobases, the risk of conflicts between identical objects is significant. In this case, use interactive mode for manual mapping of the conflicting objects.

To run the wizard, click  Synchronize with additional parameters in the Data synchronization list.

Connection parameters

On the first page of this wizard, you must select the exchange message transport type or modify the current selection. For this, click  Set up. If necessary, you can access the data exchange directory to make sure that the required files exist or view the exchange message transport parameters.

To proceed to the next page, click Next.

Data mapping

On the second page of this wizard, object mapping statistics are displayed. Before displaying this page, the application will attempt automated data mapping.

  • The automated mapping is performed for the default mapping fields, using the strictest mapping rules. We recommend that you review and verify the automated data mapping results. The list includes:
    • Data, Total accepted, pcs
    •  - object mapping problems, attention required;
    • In this application, pcs
    • Not yet mapped, pcs
    • Mapped, %
  • Click Execute mapping to preview the imported data and perform the manual object mapping in a separate window. Each manual mapping between infobase objects will be saved for further use and automatically applied during next automated mapping. When in the interactive mode, we recommend that you start with the automated mapping and then follow it up with manual mapping for objects that failed to be mapped automatically.
  • This list is not updated in real time; to view the latest data received and mapped after the list was opened, click  Refresh.
  • To import data for the selected table rows, use the All actions - Get data command. The data import option is used rarely, only for the purpose of mapping linked data tables.
  • The above commands are located in the context menu of the mapping statistics table. When called from the context menu, these commands apply to all selected table rows.

Once the mapping is complete, the rules will be saved for further use so that this page will no longer be displayed.

To proceed with data synchronization, click Next.

Data to be sent

On the third page of this wizard, you must specify data to be sent to the other application for synchronization. If you skip this step, all data modified since the last synchronization session will be sent. The radio button is set to Do not add by default; in this case, data will be sent in accordance with common settings.

  • To export additional documents, use the Add documents radio button.
    • For period field:  select a date period for documents you want to export.
    • With filter field: All documents (hyperlink) will be sent by default. Click the hyperlink to set the filter conditions: select a field to be filtered, specify Comparison type, select Comparison value. You can specify multiple filter conditions if necessary, and group them with logical operations And, Or, Not.
  • To export additional data, use the Add data radio button.
    • Click the hyperlink to select the data you want to export. The hyperlink displays Additional data not selected by default. Fill Data to be sent from the related lists and click OK. The hyperlink will change to display the list of selected data.
  • To view the sent data statistics, click the Sent data content report hyperlink.
  • Click Settings to re-use the earlier settings or save the new settings.

Click Next to start data synchronization with the selected parameters.

Data synchronization complete

On the final page of this wizard, synchronization session status will be displayed:

  • Failed to receive data, Data received successfully, etc.
  • Failed to send data, Data sent successfully, etc.

Click Detailed to access the event log, read the detailed synchronization report, and review the errors, if any.

You can set a schedule for automated data synchronization if necessary (this can be useful when the current data synchronization is not yet included in any data synchronization schedules). Click Set up.
