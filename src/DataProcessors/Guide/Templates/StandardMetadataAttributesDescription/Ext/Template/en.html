Standard metadata attributes description

For the Products catalog's Parent standard attribute a synonym "Group"' is set, as this term is more user friendly for the group, this product is included to. In all forms and reports instead of "Parent" label the "Group" label is displayed.

Similarly, the synonym for the Parent standard attribute for the Counterparties catalog is set.

Synonyms are also set for "Owner" standard attributes in some subordinate catalogs.
