Document printing

The ProductExpense document implements the parameterized command PrintBillOfLading. It is automatically included into the form of the document and the list form. For this command the multiple mode for the parameter usage is set. In the list form when this command is called, the selected rows corresponding references will be passed into its parameter. In the command handler the server function, which performs a printed forms generation is called. The spreadsheet document is returned by this function into the client application, where it is opened.

The spreadsheet document in the edit mode is demonstrated on the order for delivery printing during the product sale.
