Filling with initial data

In documents the initial filling of attributes is demonstrated.
The Vendor and Customer attributes have FillFromFillingValue property set. This leads to filling those attributes with the data, set in the list filter if the object is created from the list.

For the Kind attribute of the Products catalog the FillValue metadata property is set (product kind- Product).

In the Counterparties catalog item form in the NewSettlementAccount command the opening form data programmatic filling with the FillingValues form parameter is demonstrated. The filling handler is in the CounterpartySettlementAccounts catalog object module.
