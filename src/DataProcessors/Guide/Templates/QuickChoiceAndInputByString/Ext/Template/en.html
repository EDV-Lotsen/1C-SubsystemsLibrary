Quick choice and Input by string

The technique of overriding the standard quick choice and entering by string is demonstrated on an example in the Warehouses catalog. The algorithm is implemented in the manager module of the catalog.
For entering by string the query is added, which excludes items, having an attribute DontUse set.
For the quick choice in the list included all items, but for items with the DontUse attribute set, a warning will be displayed on item choice. Input by string is exampled in the Products and Counterparties catalogs.
