Placing pictures in the spreadsheet document cells

The capability of placing pictures in cells of the spreadsheet document are illustrated in the the price list of the Products catalog.

For the Products catalog the PrintPriceList command is implemented. The handler of this command calls the server function that performs selection of products data using a query. Then on basis of selected data the spreadsheet document is filled using the PriceListTemplate template. The spreadsheet document is returned by this function to the client application and opened.
