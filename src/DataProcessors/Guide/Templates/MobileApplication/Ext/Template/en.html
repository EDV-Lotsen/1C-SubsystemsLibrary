Mobile application

The 1C:Demo application was developed as a demonstration of 1C:Enterprise mobile platform capabilities.

The application is an autonomous workplace for a sales manager that visits customers. In the standard scenario, the mobile infobase with this application synchronizes data with the demo infobase.

The following operations are available to the sales manager:

  • enter new sales orders;
  • view previously entered sales orders;
  • enter new counterparties;
  • get warehouse balances;
  • view products and warehouse info (read-only).

The following data viewing options are available:

  • the document list with conditional formatting used to distinguish between order states;
  • the desktop form that displays sales orders entered at a specific date;
  • The Order analysis data processor that filters sales orders by price type.

The warehouse balances report is generated remotely in the master infobase. The Warehouse stock balance data processor uses the report for getting balances by specific products, by all products, in a specific warehouse, or in all warehouses. The report can be obtained immediately on a request, or during the next data synchronization.

The counterparties catalog demonstrates features specific to the mobile platform:

  • make phone calls;
  • send text messages;
  • obtain current geographical coordinates;
  • get address by geographical coordinates;
  • display location on a map by geographical coordinates;
  • create and use multimedia (photo, audio, and video files).

The Managed application demo application supports data synchronization with mobile applications. This includes:

  • Mobile exchange plan;
  • The MAExchange web service for transferring exchange messages and prepared batches;
  • MobileReports information register for storing the generated report.

Besides the application, the delivery kit includes a picture library (resources.zip) that you can use in development of demo applications for Android or iOS.

Before you start using the demo configuration:

  1. Publish the MAExchange web service for the demo infobase.
  2. Install the mobile application on a mobile device.
  3. Set up the data exchange between the mobile infobase and the demo infobase. In the settings form, enter the following:
    • published service address;
    • user account for running the service;
    • whether user name and password are required.
  4. Perform the data synchronization to get the initial infobase state.

When you start the application for the first time, it prompts you to specify the settings and synchronize data.
