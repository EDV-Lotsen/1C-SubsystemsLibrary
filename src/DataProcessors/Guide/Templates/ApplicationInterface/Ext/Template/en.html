Application interface

If the user is running multiple infobases it is difficult to determine which information base is opened in which window. To simplify the infobase window identification it is possible to specify a Short Caption for the window. The Caption can be generated automatically or set manually. If a Short Caption is set, the captions of all windows belonging to the same information base, will begin with this caption. For example, If the caption is set to "DEMO", then the Caption of the main window will be looking like this: DEMO. Demo Configuration "Managed Application" (1C: Enterprise).

In this configuration the manual way of the Short Caption specifying is used, nevertheless it fully demonstrates this feature.

To set a Short Caption open the user settings form and change the "Short Application Caption" field.
