Functional options

In the configuration several functional options are implemented.

Functional options AccountingByWarehouses and HandlingShopEquipment operate on entire the system. They are associated with the corresponding constants, which keeps values of these functional options. For various configuration objects functional options are defined in the FunctionalOptions property. For example for the Warehouses catalog, Warehouse document attributes and so on.

The Multicurrency functional option is set separately for each company. its values are stored in the Companies catalog attribute. The configuration have the Company functional options parameter, associated with the Companies catalog. The functional option includes various objects attributes, also the currencies catalog and CurrencyRates information register are included. In forms of documents where a currency is selected, a specified by the user company is set as the Company functional option parameter. This allows to determine the current value of the functional option of the opened form. The Currencies catalog and CurrencyRates information register, as well as the Currency documents lists columns are displayed if there is at least one company for which the accounting in currency is enabled.

After changing constants or the RefreshInterface method is called.

In FillCheckProcessing documents handlers the currencies checking is disabled for companys, which do not perform accouting in currency.
