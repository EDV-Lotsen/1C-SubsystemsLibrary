Access rights

In the configuration the Salesman role exists. It have only rights which are enough to create and post the ProductExpense document and to view catalogs. All unavailable objects are automatically excluded from the command interface.
For the ShipmentEvidence attribute of the ProductExpense document only view right is set therefore in the form this attribute is not enabled for editing.

In metadata of documents for which the posting is used, the PostInPrivilegedMode and UnpostInPrivilegedMode properties are set. This allows to perform reading and writing of registers during posting while those registers are not available for this user.
Also there are the Purchases Manager and the Sales Manager roles in the configuration.
