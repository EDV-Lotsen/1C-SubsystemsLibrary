Reports

Handling reports is demonstrated with several reports included in the configuration. All reports have a Data Composition Schema, which contains the full report description.

In reports the report user settings and report variants handling is demonstrated. Settings that are supposed to be edited by the user are marked as user settings.

In the SalesDynamics report two standard variants are defined.

In the WarehouseStockBalance report the command for calling the report from other forms with the specific filter is demonstrated.

In the MoneyExpenseDocuments report the usage of the link between choice parameters, defined in the configuration is demonstrated - in the filter field for the "Vendor Settlement Account" in the choice form the filter by the "Owner" field is applied automatically. Also when the "Vendor" field is changed in the filter, the "Vendor Settlement Account" will be cleared automatically not to select values for the other vendor settlement account.
