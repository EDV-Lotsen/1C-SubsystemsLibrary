If one or more working processes run on a Linux server, and one or more client computers are connected via web client, the infobase can contain files that skipped text extraction procedure and therefore cannot be found by using full-text content search. In this case, you need to start thin or thick client on any client computer running under Microsoft Windows, and open the Automatic text extraction data processor. This data processor will extract text from file contents automatically. The default execution time interval is 60 seconds.
