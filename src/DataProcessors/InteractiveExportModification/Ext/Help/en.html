This list is used to select data to be sent to another application during data synchronization.

To access this list, click a hyperlink in the Filtered field on the Data to be sent page of Interactive data synchronization wizard. 

Selecting data to be sent

  • Click Pick to select data to be sent and fill the list. Applied objects available for selection are displayed as a tree. Select the required objects and click Select. The selected data will be added to the list.
  • To delete a data item from the list, click  Delete.
  • The list of data to be sent contains the following columns:
    • Data to be sent - applied objects marked to be sent are listed here;
    • Filter - All objects are selected by default. Click the All objects hyperlink to select required items from the selected lists:
      • Click Add condition, select Field for filtering, select a comparison operation in the Comparison type column, specify Value. You can create multiple filter conditions if necessary. To apply logical operations And, Or, Not, use the Group conditions command;
    • To be sent/Total - number of selected list items and total number of list items. To recalculate the number of selected items, click the Will be sent hyperlink at the bottom of the list.
  • Click OK to confirm and return to Interactive data sychronization wizard to resume data synchronization setup.

Viewing export data content

  • To view Sent data content information, click Data content.

Saving and using data export settings

To save the current settings or load previously saved settings, click Settings.

  • To load previous settings (for example, auto-saved settings), select Last data sent from the list.
  • You can select any previously saved settings from this list.
  • You can also save your current settings. For this, click Save current settings, type in a description for the settings you want to save, and click Save.

Calculating number of objects to be sent

  • To calculate the number of objects to be sent, click the Will be sent hyperlink at the bottom of the list. The calculated number of items will be displayed in the To be sent/Total column, and the hyperlink will change to display the total amount of items to be sent.
