Connection parameters

To set up the data exchange, you need to know the address of the other infobase. To find out this address, start the application, then on the Main menu point Help and click About.

Pay attention to the following parameters (all of them can be copied from the About form):

  • Mode - file or client/server. If the infobase runs in the file mode, it is located in the directory on this computer or on the local network. If the infobase runs in the client/server mode, you need to know the server name and the name of the infobase on the server.
  • Directory - if the infobase runs in the file mode, specify the directory where it is located.
  • Server - if the infobase runs in the client/server mode, specify the 1C:Enterprise server name where the infobase is located.
  • Infobase -if the infobase runs in the client/server mode, specify the name the infobase on the 1C:Enterprise server.
  • User - name of the user on behalf of which the connection is established. If the name contains ";", use the part from the beginning to ";" as the user name.
