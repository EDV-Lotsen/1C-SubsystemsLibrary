Connection parameters

To setup the data exchange you need to know the address of the other infobase. To find out this address, start the application in the web client mode, then on the Main menu point Help and click About.

Pay attention to the following parameters (all of them can be copied from the About form):

  • Address - infobase address on the Internet.
  • User - name of the user on behalf of which the connection is established. If the name contains ";", use the part from the beginning to ";" as the user name.
