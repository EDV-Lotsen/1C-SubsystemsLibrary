Basic Functionality

The Basic Functionality subsystem includes the basic functionality for all applications that use 1C:Subsystems Library.

It allows you to implement the following features:

  • Basic procedures and functions
  • Setting application window title
  • Selecting metadata objects
  • Batch object changes
  • Deletion of marked objects
  • Form settings management
  • List of references that are not taken into account during reference-dependent operations (such as object deletion)
  • The Event Log

The Basic Functionality subsystem adds the following forms and form items:

  • Event Log form
  • Delete Marked Objects form
  • Form Settings Management form
  • Choose Metadata Objects form (allows you to select multiple metadata objects)
  • Batch Editing form (allows you to change properties of multiple objects simultaneously)
