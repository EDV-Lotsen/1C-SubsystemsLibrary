Petty Cash Expense

Subsystems used

  • Additional Reports and Data Processors
  • Basic Functionality (form settings management)
  • Companies
  • Currencies
  • E-mail Operations (the option to send a document by e-mail)
  • Object Prefixes
  • Printing (including print templates)

Other notable mechanisms

  • Filling based on another document (the Fill according to the document that the petty cash expense is baswed on button and other similar buttons)
  • Creating documents based on this document
  • Printing
  • Viewing the entire set of documents that are based on each other (the View document hierarchy button)
  • Viewing register records for a document (the View register records report button)
  • Counterparties and counterparty agreements
  • Accumulation registers
  • Accounting registers
  • The option to show or hide the list of entries, which allows to replace a single table entry with a set of form fields
