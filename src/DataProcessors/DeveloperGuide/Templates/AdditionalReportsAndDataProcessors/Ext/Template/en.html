Additional Reports and Data Processors

The Additional Reports and Data Processors subsystem allows you to implement the following features:

  • Importing reports and data processors
  • Generating reports and running data processors
  • Specifying forms where report or data processor buttons are available
  • Specifying access rights to reports and data processors

The Additional Reports and Data Processors subsystem adds the following form items:

  • Additional Reports in the Reports box
  • Additional Data Processors in the Tools box
  • Report and data processor buttons in various forms
