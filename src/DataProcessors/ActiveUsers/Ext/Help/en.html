The list contains the list of user that currently work with the application.

You can sort the list by any column. With the bold font the current session that opens the active users list is highlighted.

Using commands in the All actions menu you can adjust the form and export the list content as a spreadsheet or text document. Using the Event log command you can open the event log filtered by the selected user.

In the Client-server mode the Terminate session command is available. This command forces shutdown for the selected user sessions. For correct operation please specify the infobase administration parameters (the All actions menu).
