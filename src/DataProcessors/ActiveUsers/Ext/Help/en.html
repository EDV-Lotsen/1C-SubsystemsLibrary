The list of active users provides information on users who are currently connected to the infobase.

You can sort the list by any column. The row corresponding to the current session is highlighted with bold formatting.

You can setup the form and output the list into a spreadsheet or text document with All actions menu. To go to the event log filtered by the selected user, click Event log.

You can terminate user sessions by clicking Terminate session if the application runs in the client/server mode. User termination requires the infobase administration parameters to be set up (see the "Infobase administration parameters" command of the All actions menu).
