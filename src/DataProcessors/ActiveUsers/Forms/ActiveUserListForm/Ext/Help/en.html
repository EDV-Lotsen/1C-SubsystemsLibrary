The list of active users provides information on users who are currently connected to the infobase. Not only users that are connected with the "Thin client", "WebClient", or "Designer" applications, but also "COM connection", "WS connection", "Background job", and so on are shown in the list. 

For analyzing user activity, go to event log: select the user in the list, and then click "Event log".

If infobase runs in the client/server mode, you can terminate a user session by clicking "Terminate session". User termination can require additional 1C:Enterprise server connection parameters to be set up (see the "Infobase administration parameters" command of the "All actions" menu).
