In case of file mode it is possible to set the automatic execution for scheduled jobs.

Property name A Property purpose
Automatically open a separate session for execution of scheduled jobs +

When this check box is set the separate session for execution of scheduled jobs will be opened in one second after the main application session is opened.

If the application was started with scheduled job execution option, the separate session for execution background jobs will not be started as the current session will be used for that.

A separate session is opened on behalf of the current user and procedures are executed in privileged mode.

Notify if scheduled jobs are not executing Notify a user that scheduled jobs are not executing when Automatically open a separate session for execution of scheduled jobs check box is set.
Notification period, minutes The period in minutes between checking states of scheduled jobs execution and notification of a user in case of errors.

Note: A column contains + sign if the property can be changed only by Administrator.

Command Details
Open separate session.

For file mode only. Opens a separate session for execution of scheduled jobs. The session is opened with options that allow it to process background jobs and when the current session is closed, this separate session will be closed automatically too.

 

Infobase startup options to enable automatic background jobs execution

Parameter

 Details
ExecuteScheduledJobs Execute scheduled jobs in this session.
IgnoreWarnings Do not show warnings if the session for background job execution is opened already or there are other reasons why scheduled jobs can not be executed in this session.
SeparateSession Open the session only for execution of scheduled jobs. If the session for this purpose is started already, the currently opening session will be closed with warning (or without warning, see IgnoreWarnings option).

Examples for Windows command prompt

Purpose

 String
Enable execution of scheduled jobs in currently opening session. "C:\Program Files\1cv82\Bin\1cv8c.exe" /CExecuteScheduledJobs

Open a session only for execution of scheduled jobs. For example, if infobase directory is located on a separate computer, this string can be used in Windows scheduler. Excessive sessions will be closed automatically.

 "C:\Program Files\1cv82\Bin\1cv8c.exe" ENTERPRISE IBConnectionString"File=""C:\Infobase"";" /NJohn /PPassword /C"ExecuteScheduledJobs IgnoreWarnings SeparateSession".
