This form is used to set up scheduled jobs for performance score export.

This functionality may be useful when you need to run performance analysis for an external application.

To access the form, execute the Settings command located in the Performance monitor window.

Setting up automatic performance score export

  • Select the Execute export check box; this will automatically select the Execute export to local directory and Execute export to FTP check boxes.
  • Fill in path in the Local export directory field (clear the corresponding check box if export to a local directory is not performed);
  • specify FTP server address if necessary, or clear the Execute export to FTP check box.
  • To proceed with export setup, click Set export schedule and specify the export schedule.
  • Click Save and close to confirm and go back to the application.

Once the schedule is set up, performance score export will be performed automatically. The generated external files, as well as files with application settings used for performance testing, are compressed to ZIP archive automatically.
