To generate a report, click Generate. The report lists the unassigned tasks that cannot be performed because of the empty list of roles and task performers.

To view only tasks assigned to performers for addressing objects supervised by a specific Task control manager, use the Task control manager filter.

To group the unassigned task list by addressing object, select the Unassigned tasks report variant.

To group the unassigned task list by performer role, select the Unassigned tasks by role report variant.

To view the number of unassigned tasks broken down by performer role, select the Unassigned tasks by addressing object report variant.

To view both the unassigned and overdue tasks, select the Overdue tasks report variant.

To access detailed report settings, execute the All actions - Change variant command.
