Viewing the full user rights list.

To access this report, execute the Access rights report command located on the Access rights tab of the user or external user (user group or external user group) card.

The report can be printed or saved to file in any available formats using the standard commands.

The report displays the following informations:

  • Infobase user properties:
    • infobase access, login name, OS user name, and more;
  • User access groups:
    • Group name, responsible person, profile;
  • User roles by profile;
  • Viewing objects:
    • This section is a table with 3 columns: Object/Rights/Restrictions, Access group/Profile/Roles, Restrictions on record level/Access types and values;
  • Editing objects:
    • This section is a table with 3 columns: Object/Rights/Restrictions, Access group/Profile/Roles, Restrictions on record level/Access types and values;
  • Object access rights: Directories.

Generating detailed reports

Select the Access rights details check box and click Generate.
