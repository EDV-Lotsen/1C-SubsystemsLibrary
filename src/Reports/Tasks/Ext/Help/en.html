To generate the report, select a variant and click Generate.

Current tasks

The following report parameters can be set if necessary:

Author Show tasks by this author only
Completion date Show tasks with this completion date only
Performer Show tasks assigned to this performer only
Due date Show tasks with this due date only
Creation date Show tasks with this creation date only
Completed Show completed tasks only (or uncompleted tasks only)

 

Performance discipline report

By default, all tasks performed during the current month are displayed in the report.

To access detailed report settings, execute the All actions - Change variant command.
